Step 1: Fixed window

👉 Learn:

Step 2: Prefix sum basics
Build prefix array
Range sum queries

👉 Learn:

sum(i,j) = prefix[j+1] - prefix[i]



Step 3: Prefix + HashMap (VERY IMPORTANT)

This is where most interview problems come from.

Problems:

Count subarrays with sum k
Longest subarray with sum k

👉 Core idea:

If (prefixSum - k) exists → we found a subarray


Step 4: Variable sliding window

Smallest subarray ≥ k
👉 Learn:

Step 5: Kadane’s Algorithm

👉 Core idea:

If sum < 0 → reset
